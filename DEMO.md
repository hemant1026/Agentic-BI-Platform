# Demo Guide

## Live Demo Walkthrough

This guide walks through a complete demo of the Agentic BI Platform, showing the full analysis pipeline from file upload to actionable insights.

---

## Demo 1: Retail Sales Analysis

**Dataset**: `data/sample_data.csv` (20 rows, 6 columns)

### Step 1: Upload

1. Open http://localhost:3002
2. Click "Choose File" and select `data/sample_data.csv`
3. Click "Load & Analyze Dataset"

### Step 2: KPI Dashboard

The system automatically generates KPIs:

| KPI | Value | What It Tells You |
|-----|-------|-------------------|
| Sales Mean | $13,935 | Average daily sales |
| Sales Max | $25,000 | Best sales day |
| Quantity Mean | 76.5 | Average units sold per day |
| Price Median | $280 | Typical price point |

### Step 3: Visualizations

The system auto-generates 4-5 charts:

- **Distribution of Sales**: Histogram showing sales distribution across days
- **Frequency: Region**: Bar chart showing order count by region (North/South/East/West)
- **Sales vs Quantity**: Scatter plot revealing the price-volume relationship
- **Correlation Matrix**: Heatmap showing relationships between all numeric columns

### Step 4: Chat

Open the chat drawer and ask:
- "Which region has the highest sales?"
- "Is there a correlation between price and quantity?"
- "What's the trend in sales over time?"

## Demo 2: E-commerce Deep Dive

**Dataset**: `data/ecommerce_orders.csv` (100 rows, 13 columns)

### What to Show

1. **Multi-column KPIs**: The system generates KPIs for total_amount, quantity, unit_price, shipping_cost, and discount_pct simultaneously
2. **Category Analysis**: Auto-detects product categories (Electronics, Apparel, Home, Sports, Books) and generates frequency charts
3. **Payment Method Distribution**: Shows Credit Card vs PayPal vs Debit Card breakdown
4. **Regional Comparison**: Four-region analysis with balanced distribution

### Key Talking Points

- "The system detected 5 numeric columns and 8 categorical columns automatically"
- "It chose bar charts for categorical data and histograms for continuous data"
- "The correlation heatmap shows quantity vs unit_price relationship"

## Recording a Demo Video

### Setup

```bash
# Start the application
cd backend && node server.js &
cd frontend && npm start &

# Wait for both servers to start
# Open http://localhost:3002 in Chrome
```

### Recording Tips

1. Use a clean browser window (no bookmarks bar, no extensions)
2. Set browser zoom to 100%
3. Use a screen recording tool (OBS, Loom, or QuickTime)
4. Record at 1920x1080 resolution
5. Keep the demo under 3 minutes for the video
6. Upload the sample datasets in order: sample → ecommerce → financial

### Post-production

- Add captions for key moments
- Speed up the loading animations (2x)
- Add a title card: "Agentic BI Platform - Live Demo"
- Export as MP4, upload to YouTube (unlisted), and link in README
